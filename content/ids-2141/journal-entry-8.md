Using Fusion360 was a lot easier than using FreeCAD. Everything feels more intuitive and creating the shapes was way more obvious. It seems that everything in FreeCAD is parametric, while in Fusion the user has the option to do things in a similar fashion to how I'd sketch a drawing.
